Github projects by Michel Stuyts

  • QuickCRS: A QGIS plugin to enable your favourite CRS (OTF).
  • Plugin Load Times: A plugin to show how long each QGIS plugin loads when QGIS starts.
  • phpQGISrepository: A simple PHP script to create a repository for QGIS Python plugins.
  • Belgium WMS: A list of WMS services with data for Belgium.
  • Belgium WFS: A list of WFS services with data for Belgium.
  • Belgium WMTS: A list of WMTS services with data for Belgium.
  • Belgium-TileMap: Tilemaps for Belgium.
  • Delicious2MySQL: A PHP script to backup your del.icio.us links to a MySQL database.
